Este es otro servicio de ejemplo orientado a desarrollo técnico.

La plantilla permite:

- Ofrecer reservas pagadas por adelantado
- Integrar pagos en Bitcoin, Lightning o fiat
- Liquidar siempre en Bitcoin para el comerciante
- Configurar calendarios múltiples
- Gestionar disponibilidad avanzada

El objetivo es mostrar cómo un desarrollador o agencia puede utilizar esta plantilla como:

- Sistema de reservas
- Sistema de facturación
- Sistema de ecommerce
- Infraestructura de pagos Bitcoin-first

La estructura es completamente personalizable y lista para producción.
